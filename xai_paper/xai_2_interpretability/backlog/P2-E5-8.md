Method (Chan et al. 2022): behaviour = the true-graph SINK cell(s); TRUE hypothesis =
{outputs} ∪ transitive-closure ANCESTORS in the A1 ground-truth graph; WRONG = same size
with a genuine (live) ancestor dropped. Resample-ablate every out-of-circuit cell with the
oracle do-set {0,base+17,base+37} (the values that established the edges) and re-run the
real ROM `horizon` frames; PASS iff behaviour preserved over the whole resample
distribution. Run command: `julia --project=<primary>/jutari
tools/xai_study/phaseC_mechanistic/causal_scrubbing.jl --games core`. Self-check
(`--selftest`, Pong positive control) passes.
