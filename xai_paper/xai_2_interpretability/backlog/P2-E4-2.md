RESULT (jutari, 6 core games, f120+30, NOOP trace):
- CONTENT output (a non-zero, causally-persistent candidate RAM cell read one-hot;
  ∂y/∂u alive) = the FAITHFUL headline. DeepLIFT corr / precision@3:
  pong 0.799/1.0 · breakout 0.75/1.0 · seaquest 0.813/1.0 · ms_pacman 0.822/1.0 ·
  qbert 0.999/1.0  (FAITHFUL: the content byte is its own top cause; Grad×Input/
  DeepLIFT land their top RAM byte exactly on it). Space Invaders 0.215/0.33 is the
  honest UNFAITHFUL case: the local one-hot gradient lands on the read byte (bullets,
  ram[81]) but the TRUE dominant cause over the horizon is the INDIRECT ram[17]
  (invaders_left) — "local gradient ≠ global cause". DeepLIFT completeness err = 0
  (rescale rule is exact on the linear one-hot read). Grad×Input ≡ DeepLIFT ≡ vanilla
  saliency in RANKING here (one-hot read ⇒ same single non-zero entry); DeepLIFT
  uniquely adds completeness.
- POSITION output (a discriminating sprite-pixel cell) = the "plausible ≠ faithful"
  contrast: the content-path gradient VANISHES (corr=0, p@k≈chance) on ALL 6 games
  (the §1 index failure), while the intervention oracle has real signal — so the
  oracle is the sole truth there.
- HARNESS POSITIVE CONTROL passes on all 6 games / both outputs: feeding the oracle's
  own |Δy| as the candidate map scores corr=1, precision@k=1 (the metric rewards a
  faithful map). Self-check asserts this + the per-game vanishing/faithful behaviour;
  bit-exact re-run asserted before scoring; nothing fabricated.
