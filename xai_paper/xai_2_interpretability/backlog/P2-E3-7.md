**Result (the quantified-Kording lesson).** Matched-component fraction is driven by
KNOWN SIGNALS (clock / R-W activity / vsync phase), NOT game variables: across the
core set the top latent components overwhelmingly match the chip's frame cadence,
not causal game state. NMF matched-fraction (mean ≈0.6) > PCA (≈0.6 but
clock-dominated) — NMF's non-negative parts are slightly more single-variable but
still surface bookkeeping, not causation. Per-cell recovery F vs the exact oracle is
LOW/negative on most games (variance ≠ causal role; present ≠ used), confirming
dim-reduction is a weak interpretability method despite fully-known structure. See
`out/A7_dimred_all.json` for the PCA-vs-NMF roll-up.
