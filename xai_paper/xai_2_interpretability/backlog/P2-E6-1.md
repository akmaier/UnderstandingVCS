## Goal
Read the append-only `results_index.csv` (SPEC §R) and place every Phase-A/B/C method on the
two shared reporting axes — faithfulness (X, vs the §1 oracle ground truth) vs human-plausibility
(Y) — producing one comparable F∧S(∧M-where-available) score per method plus a discrepancy table
that highlights the danger zone (high plausibility, low faithfulness). This is the analytical core
the headline figure (E7) reads from; it must be a pure read over the results index (no re-running
experiments).

## Done (P2 S6)
`tools/xai_study/compare/leaderboard.py` — pure read over all 257 committed per-game §R
records across Phase A/B/C `out/*.json` (summaries/`_all`/`core_summary` excluded), folded
into 31 method rows + the oracle positive control. Runs without `results_index.csv` (it
globs the phase `out/` trees, which ARE the index; uses the index only as an optional
source list if present). Outputs: `compare/out/leaderboard.{json,csv,md}`.

- **Faithfulness (X)** = per-record `extra.triad.F` (oracle-oriented) where present, else
  the record `value` re-oriented to faithfulness-vs-oracle via a documented per-metric
  `FAITHFULNESS_ORIENT` map (corr/F1 clipped to [0,1]; error/leakage rates one-minus'd;
  exact-recovery error → 1 iff 0). Correlation legs clipped to [0,1].
- **Plausibility (Y)** = a TRANSPARENT method-tradition PROXY (`PLAUSIBILITY_BY_TRADITION`,
  grounded in §7's matrix), clearly labelled `plausibility_proxy` everywhere — not a
  measurement (no human-rating study in P2). It exposes the §0 danger zone.
- **Headline:** on the position/index regime causal/intervention faithfulness **0.41**
  (n=4) vs gradient/correlational **0.07** (n=9), gap **0.34**; all-regime **0.68** vs
  **0.29**, gap **0.39**. Pure-gradient methods (saliency, IG, SmoothGrad, GuidedBP,
  Grad×Input/DeepLIFT, Expected-Gradients) = **0.0** faithfulness on positions (naive
  gradient vanishes, §1) while occlusion/counterfactual/extremal-perturbation + activation
  patching recover them. Top danger-zone: expected_gradients (Δ0.87), linear_probing
  (Δ0.69), A3_tuning (Δ0.68), A6_granger (Δ0.66).
- **Self-check (5/5 PASS):** every expected method present; no null faithfulness; positive
  control present and == 1; position-regime gap > 0; plausibility proxy ∈ [0,1].
