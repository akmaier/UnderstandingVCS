## Result (P2 S5)
Implemented JULIA (jutari real-ROM substrate, mirroring das.jl/activation_patching.jl —
NOT the python skeleton path; SM directed `linear_probing.jl` → `out/linear_probing_*`).
Linear probing + control tasks (Alain & Bengio 2017; Hewitt & Liang 2019) over the
recorded 600-frame RAM trajectory of each core game (seeded varied actions so the state
is rich/non-collinear and selectivity is meaningful). Per labelled concept cell: a
closed-form one-vs-rest ridge LINEAR probe decodes the cell's 4-bin value from the REST
of RAM (cell held out) → probe acc (PRESENT); a CONTROL task (random label per raw
value-type) → control acc; selectivity = probe − control. Cross-referenced to the exact
intervention oracle (das.jl `transient_downstream_cells`) to count DECODABLE-BUT-NOT-CAUSAL
cells — the present-vs-used gap (§7 "Partial / misleading — the tuning-curve trap").

Ran SYNC over all 6 core games + Pong self-check (positive control: ≥1 decodable cell,
mean selectivity>0, probe≥majority floor). Headlines (probe / selectivity / present≠used):
