### DONE (P2 S5) — what was built
Written in **Julia** (SCRUM §5/§7 — experiments run on JUTARI locally), not the `.py`
placeholder named in `file_scope`; the runner is
`tools/xai_study/phaseB_attribution/ig_baseline_sweep.jl` and its artifacts are
`tools/xai_study/phaseB_attribution/out/ig_baseline_sweep_*`. It reuses the IG-pilot scorer
(pearson/spearman/precision_at_k, deletion/insertion AUC on the TRUE VCS, the per-cause |attr|
map, the oracle-as-method positive control) and the bit-exact oracle/Cause machinery
verbatim, mirroring the gradxinput/smoothgrad siblings (same 6-core-game contract,
per-game RomSettings/ROM-alias map, content-vs-position output split).

Run: `julia --project=<repo>/jutari tools/xai_study/phaseB_attribution/ig_baseline_sweep.jl
--games core --baseline-sweep` (self-check: add `--selftest`). 6 core games, 2 outputs each
(content one-hot RAM read = headline; ball_pixel = position contrast), full pass.

**Baseline sweep (the §5 finding), 4 baselines:** zeros · mean_ram · random · real_alt
(genuine RAM at an earlier frame = on-distribution reference). Per-game content corr(zeros)
[range over baselines], precision@1, completeness err≈0 everywhere; oracle-as-method control
corr=1/p@1=1 on every non-degenerate column:
- qbert  ram[39] corr 0.999 [0.000..0.999] Δ=0.999 p@1=1
- ms_pacman ram[16] corr 0.811 [0.000..0.811] Δ=0.811 p@1=1
- breakout ram[30] corr 0.750 [0.000..0.750] Δ=0.750 p@1=1
- pong   ram[54] corr 0.407 [0.407..0.407] Δ=0.000 p@1=1
- space_invaders ram[81] corr 0.215 [0.000..0.215] Δ=0.215
- seaquest ram[118] corr 0.000 [0.000..0.019] Δ=0.019 (content byte=0 at the state)
- position (ball_pixel): IG max|attr|=0 for ALL baselines, all 6 games → near-chance
  (plausible ≠ faithful; no baseline manufactures a missing gradient).

**Sharper-than-textbook result:** on the one-hot content path the IG *magnitude* is
baseline-sensitive (the (x−x0) factor; ig_magnitude_sensitivity per game) but the
faithfulness *corr/ranking* is INVARIANT among non-degenerate baselines (∂y/∂u=e_idx is
constant) — EXCEPT the DEGENERACY: a baseline byte equal to the content byte kills IG there.
Empirically the `zeros` baseline is degenerate when the content byte is 0 (seaquest), and
`real_alt` is degenerate when the concept byte is unchanged frame-to-frame (5/6 games) — both
collapse corr to 0. `n_baselines_ig_alive` + `degenerate_baselines` flag it per game.
