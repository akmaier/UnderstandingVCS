## Goal
Stand up the Phase-B pilot: run Integrated Gradients on a single chosen output `y`
(e.g. Pong ball-pixel or score) for one game and compare the attribution map against the
ground-truth intervention oracle (E1-1) and gradient oracle (E1-2). This validates the
attribution harness end-to-end (output selection, baseline choice, scoring against the
oracle) before the full method fan-out in E4-1..E4-13.
