RESULT (N=500 masks, keep_prob=0.5, f120+30): RISE is a black-box perturb-and-re-run
saliency — N random binary masks over the candidate RAM cells, mask-to-0 (= the oracle's
own occlude!), re-run the real ROM, importance(i)=(1/(N·p))·Σ y·m[i] (Petsiuk eq.). Budget
= exactly N real re-runs per output. Content-path corr vs oracle: qbert 0.98, ms_pacman
0.79, breakout 0.73, pong 0.64, space_invaders 0.35, seaquest 0.0 (flat — Generic settings,
no signal at this window; honest). Convergence corr STABILISES as N grows (e.g. qbert
0.89→0.98 over 62→500), confirming RISE variance ∝ 1/N. POSITION contrast: RISE still
attributes on ball_pixel where IG/saliency VANISH (qbert 0.55, space_invaders 0.47). Harness
positive control (oracle-as-method corr=1/p@k=1) holds on every non-degenerate column.
Cluster-shardable: `--shard i --nshards n --shard-kind game` + `--out-dir` (tools/cluster/
xai_array_jl.sbatch).
