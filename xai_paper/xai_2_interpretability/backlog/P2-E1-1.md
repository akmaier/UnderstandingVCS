## Goal
Build the primary ground-truth oracle: for a VCS output `y` and candidate cause `u`,
intervene (occlude / clamp / replace / resample), re-run deterministically, and record
Δ`y`(u) as the exact causal map. Batched via SOFT-STE (bit-exact to HARD). This is the
reference object every Phase A/B/C method is scored against.
