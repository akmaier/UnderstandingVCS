A bright cell (row = cause, column = effect) is an edge; the difference between the two panels is the error. discovered_circuit_best_F1_vs_true_dataflow = 0.000 — this example only (pong, state f30+30); the audit aggregate is below.

What it does

Prunes edges to find a minimal circuit; scored by edge precision/recall and scrubbing-preserved performance against the true data-flow.

How it's scored

Best F1 of the auto-discovered circuit vs the true data-flow over a threshold sweep. 0.00 — the pruned graph does not match the true edges here.

The score is measured against the §1 intervention oracle — never against another interpretability method. F (faithful) is always vs the oracle; see the execution stack.
