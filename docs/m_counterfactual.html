What it does

Finds the minimal valid on-distribution edit that flips the output — set-state-and-re-render removes the off-manifold objection.

How it's scored

Pearson correlation / minimality of the minimal valid counterfactual edit against the oracle's minimal causal set. Set-state-and-re-render finds nearly the true causes — its attributed region visibly matches the oracle's on the frame.

The score is measured against the §1 intervention oracle — never against another interpretability method. F (faithful) is always vs the oracle; see the execution stack.
