Each point is one intervention site: its exact causal effect from the oracle (x) against the method's recovered/approximate effect (y). Points on the dashed diagonal mean the method recovered the true effect. max_abs_recovered_minus_exact = 0.000 — this example only (pong, state f120+30); the audit aggregate is below.

What it does

Patches a recorded state value from one run into another and measures the causal effect, comparing it to the exact patch effect from the oracle.

How it's scored

Maximum absolute difference between the recovered patch effect and the EXACT patch effect from the oracle. 0.00 — activation patching is exact by construction, because it is an intervention.

The score is measured against the §1 intervention oracle — never against another interpretability method. F (faithful) is always vs the oracle; see the execution stack.
