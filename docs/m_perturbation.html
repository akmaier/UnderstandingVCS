What it does

Learns the minimal mask that most changes the output; scored by IoU with the true causal set.

How it's scored

Pearson correlation / mask-IoU with the oracle's true causal set. 0.45 — the learned minimal mask partly overlaps the true causes.

The score is measured against the §1 intervention oracle — never against another interpretability method. F (faithful) is always vs the oracle; see the execution stack.
