What it does

Averages the gradient over input noise to denoise the saliency map.

How it's scored

Pearson correlation with the oracle. Averaging input noise sharpens saliency on content outputs a little (the example), but cannot create a gradient where there is none — on discrete position outputs it is exactly 0 — so the aggregate stays low.

The score is measured against the §1 intervention oracle — never against another interpretability method. F (faithful) is always vs the oracle; see the execution stack.
