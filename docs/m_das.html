Each point is one intervention site: its exact causal effect from the oracle (x) against the method's recovered/approximate effect (y). Points on the dashed diagonal mean the method recovered the true effect. interchange_accuracy_aligned = 1.000 (game: pong, state f120+30)

What it does

Distributed Alignment Search aligns a learned subspace to a causal variable and measures interchange accuracy against the true variable.

How it's scored

Interchange accuracy of the aligned subspace against the true variable. 1.00 — DAS aligns to the causal variable exactly on this known circuit.

The score is measured against the §1 intervention oracle — never against another interpretability method. F (faithful) is always vs the oracle; see the execution stack.
