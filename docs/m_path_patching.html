What it does

Patches along specific paths to recover the circuit; scored by circuit precision/recall against the true routine.

How it's scored

F1 of the recovered path/circuit against the true routine's data-flow. 0.00 — the single-frame path search recovers none of the true routine at this state.

The score is measured against the §1 intervention oracle — never against another interpretability method. F (faithful) is always vs the oracle; see the execution stack.
